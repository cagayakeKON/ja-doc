# 1. 形容词作谓语和定语 (`い形容词`)

前一句是形容词作谓语的表达形式,后一句是形容词作定语的表达形式。

形容词是表示事物性质和状态的词。日语形容词都以「<ruby>い<rt>い</rt></ruby>」为词尾。形容词的词尾可以变化,一般称这种变化为“<ruby>活用<rt>かつよう</rt></ruby>”(活用)。

<ruby>形容词<rt>けいようし</rt></ruby>作谓语时的形态叫“<ruby>终止形<rt>しゅうしけい</rt></ruby>”(终止形)。由终止形后续「<ruby>です<rt>です</rt></ruby>」构成敬体句。敬体句语气恭敬,有礼貌。日语形容词还可以直接修饰名词作定语。形容词修饰名词时的形态叫“<ruby>连体形<rt>れんたいけい</rt></ruby>”(连体形)。形容词的连体形和终止形相同。

*   <ruby>学校<rt>がっこう</rt></ruby>は<ruby>遠<rt>とお</rt></ruby>いですか。(学校远吗?)
*   この<ruby>部屋<rt>へや</rt></ruby>は<ruby>大<rt>おお</rt></ruby>きいですね。(这屋子真大啊。)
*   <ruby>日本語<rt>にほんご</rt></ruby>の<ruby>教室<rt>きょうしつ</rt></ruby>はあの<ruby>新<rt>あたら</rt></ruby>しい<ruby>建<rt>た</rt></ruby>てものです。(日语专业的教室是那幢新大楼。)
*   あの<ruby>山<rt>やま</rt></ruby>は<ruby>低<rt>ひく</rt></ruby>いです。(那座山是低的。)

## 二、･･･は･･･にいます (`います` 动词)

<ruby>学生<rt>がくせい</rt></ruby>は<ruby>朝<rt>あさ</rt></ruby>、<ruby>教室<rt>きょうしつ</rt></ruby>にいます。

这个句型与第三课学习的「･･･は･･･にあります」(･･･是在･･･)不同的是:「います」表示的是有生命物体诸如人和动物等的<ruby>存在<rt>そんざい</rt></ruby>。否定形式是「･･･は･･･にいません」。

*   <ruby>先生<rt>せんせい</rt></ruby>は<ruby>体育館<rt>たいいくかん</rt></ruby>にいます。(老师在体育馆。)
*   <ruby>李<rt>り</rt></ruby>さんは<ruby>図書館<rt>としょかん</rt></ruby>にいます。(小李在图书馆。)
*   <ruby>鲁<rt>ろ</rt></ruby>さんは<ruby>閲覧室<rt>えつらんしつ</rt></ruby>にいません。(小鲁不在阅览室。)

## 三、･･･に･･･がいます (`います` 动词)

<ruby>池<rt>い</rt></ruby>の<ruby>中<rt>なか</rt></ruby>にはきれいな<ruby>金魚<rt>きんぎょ</rt></ruby>がいます。

这个句型表示“(某处)有(某人或某个有生命物体)”。「います」在这儿表示“有”。

*   <ruby>椅子<rt>いす</rt></ruby>の<ruby>上<rt>うえ</rt></ruby>に<ruby>猫<rt>ねこ</rt></ruby>がいます。(椅子上有只猫。)
*   <ruby>部屋<rt>へや</rt></ruby>に<ruby>誰<rt>だれ</rt></ruby>がいますか。(屋里有人吗?)
*   <ruby>閲覧室<rt>えつらんしつ</rt></ruby>に<ruby>学生<rt>がくせい</rt></ruby>がたくさんいます。(阅览室有很多学生。)
