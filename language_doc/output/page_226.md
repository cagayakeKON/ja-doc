# 五、格助词「に」表示动作、作用的结果

初めに縦半分に切ってください。

格助词「に」接在体言后面可以表示动作、作用的结果。

*   大学祭には牛どんを作ることに決めました。(決定校园艺术节时做牛肉盖浇饭。)
*   りんごを四つに切りました。(把苹果切成了四块。)

# 七、それほど･･･(后接否定)

牛どんの作り方はそれほど<ruby>難<rt>むずか</rt></ruby>しくないでしょう。

「それほど」是副词,表示“那样的(程度)”。「それほど」后接否定时表示并非所讲的那样,并没有达到那样的程度。相当于汉语的“并不那么……”、“不怎么……”。

*   <ruby>近<rt>ちか</rt></ruby>ごろそれほど<ruby>忙<rt>いそが</rt></ruby>しくありません。(最近不怎么忙。)
*   メモはそれほど<ruby>詳<rt>くわ</rt></ruby>しくありません。(笔记不怎么详细。)
*   それほど<ruby>好<rt>す</rt></ruby>きではありません。(不怎么喜欢。)

# 八、用言、助动词终止形+と

長く<ruby>煮<rt>に</rt></ruby>ると<ruby>固<rt>かた</rt></ruby>くなります。

「と」是接续助词,接在用言、助动词终止形后面表示假定、确定、恒常等条件,一般不能用命令等表示意志的内容结句。

表示假定条件时,「と」的前面部分可以是已经实现的事实,也可以是尚未实现的事情。「と」的后面部分经常是尚未实现的事情。
