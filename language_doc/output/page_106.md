## 三、例のように言いかえなさい。

* **例**: この花は赤いです。この花は大きいです。→ この花は赤くて大きいです。
    1. 料理はおいしいです。料理は安いです。
    2. この建物は新しいです。この建物はりっぱです。
    3. 寮は近いです。便利です。
    4. 夏は暖かいです。冬は寒いです。
    5. この映画はおもしろいです。おおぜいの人が見ます。

* **例**: 公園は静かです。公園はきれいです。→ 公園は静かできれいです。
    1. 魚は新鮮です。魚はおいしかったです。
    2. 秋の夜は静かです。秋の夜は涼しいです。
    3. 生活は楽です。生活はたのしいです。
    4. 祭りはにぎやかです。おもしろかったです。
    5. 人々は親切です。とても楽しい旅行でした。

* **例**: ホテルの部屋はきれいで広かったです。部屋代は高かったです。
    → ホテルの部屋はきれいで広かったですが、部屋代は高かったです。
    1. あしたは日曜日です。休みません。
    2. この魚は高いです。とてもおいしいです。
    3. 郊外です。あまり静かではありません。
    4. もう四月です。まだ寒いです。
    5. 李さんの家へ行きました。かれはいませんでした。
