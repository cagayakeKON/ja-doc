# 八、终助词「ね」表示确认或感叹

あのたてものはしょくどうですね。
このたてものはなんですか。りっぱですね。

终助词「ね」接在句末可以表示讲话人对事物已有相当程度的了解,只是要求对方给予确认或表示赞同(如前一例句);「ね」也可以表示感叹(如后一例句)。口语中有时把表示感叹的「ね」拉长成「ねえ」。「ねえ」的感叹语气比「ね」强。

*   これはあなたの<ruby>じしょ<rt>じしょ</rt></ruby>ですね。(这是你的辞典吧。)
*   キャンパスはとても<ruby>静<rt>しず</rt></ruby>かですね。(校园真安静啊。)

# 九、指示连体词「この」「その」「あの」「どの」

「この」「その」「あの」「どの」是一组指示连体词,用于指示特定的事物。「この」「その」「あの」「どの」不能单独使用,必须后续事物的名称。它们与说话人、听话人之间的关系和「これ」「それ」「あれ」「どれ」相同。

*   この<ruby>地図<rt>ちず</rt></ruby>はあたらしいですか。(这张地图是新的吗?)
*   あなたの<ruby>本<rt>ほん</rt></ruby>はそのつくえのうえにあります。(你的书在那张书桌上。)
*   <ruby>図書館<rt>としょかん</rt></ruby>はどのたてものですか。(图书馆是哪幢楼?)

# 十、数词、时间名词的副词性用法

说明动作、性质、状态的程度、范围、时间、方式的句子成分叫做状语。数词、副词和表示时间的名词可以直接作状语。这种表达方式我们称为数词、时间名词的副词性用法。

*   <ruby>教室<rt>きょうしつ</rt></ruby>につくえが<ruby>九<rt>ここの</rt></ruby>つあります。(教室里有九张桌子。)
*   <ruby>学生<rt>がくせい</rt></ruby>は<ruby>朝<rt>あさ</rt></ruby>どこにいますか。(学生早晨在哪儿呢?)
*   きれいな<ruby>花<rt>はな</rt></ruby>が<ruby>沢山<rt>たくさん</rt></ruby>あります。(有许多美丽的花。)
