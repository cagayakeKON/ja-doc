# 練習

一、次の文を読んでください。

1.  昨日はよい天気でした。呂さんは李さんといっしょに出かけました。
    まず<ruby>淮海路<rt>わいかいろ</rt></ruby>の一番大きいデパートで買い物をしました。それから、近くの映画館でおもしろい映画を見ました。昼食はレストランでしました。
2.  学校の食堂の料理はあまり高くありません。そして、とてもおいしいです。
3.  わたしの部屋は大きいです。しかし、あまりきれいではありません。

二、適当な言葉を入れなさい。

1.  きのうどこ○へ出かけましたか。
2.  留学生の木村さん○いっしょに買物をしました。
3.  この店は<ruby>淮海路<rt>わいかいろ</rt></ruby>○一番大きいです。
4.  そこの品物はあまり安○○○○○○。
5.  李さんと二人○食事をしました。
6.  学校の近くの公園はきれい○○○○○○。
7.  <ruby>飛行機<rt>ひこうき</rt></ruby>は何時に上海○着きますか。

三、例のように言いかえなさい。

*   例: 食事のあと散歩します。→事のあと散歩しました。

    1.  食事のあと木村さんとお茶を飲みます。
    2.  二人は近くのレストランで昼食を取ります。
    3.  公園でたくさんの写真を撮ります。
    4.  日曜日に日帰り旅行をします。
    5.  駅で切符を買います。

*   例: 国際ホテルは高いです。→国際ホテルは高くありません。

    1.  駅は近いです。
    2.  肉まんじゅうはおいしいです。
    3.  この映画はおもしろいです。
    4.  これはめずらしいです。
    5.  旅行は楽しいです。
