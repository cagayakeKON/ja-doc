# 单元三复习

* 私は上海で生まれました。→上海、私はここで生まれたのです。
* 私の誕生日は五月五日です。→五月五日、この日は私の誕生日です。

## ④挿入語

挿入語は挿在句中作补充说明的句节,它与前面的句节没有直接联系。

* 日本人は、例外もあるが、すしが好きだ。

## 三、助词

| 种类           | 用法       | 例句                                                      |
| :------------- | :--------- | :-------------------------------------------------------- |
| 格助詞「より」 | 比較対象   | テニスはバドミントンより難しいです。                      |
| 格助詞「と」   | 称谓内容   | わたしの名前は王友民と言います。                          |
| 准体助词「の」 | 替代省略   | この本はわたしのです。                                    |
| 接續助詞「から」 | 原因理由   | 注意事項を言いますからよく聞いてください。                |
| 接續助詞「ので」 | 原因理由   | あしたは日曜日なので休みです。                            |
| 副助詞「だけ」 | 限定       | 正しい答えは一つだけです。                                |
| 并列助詞「とか」 | 列举       | 訓読みでは、「さかな」とか、「うお」とか読みます。          |

## 四、助动词

| 种类   | 用法 | 例句                                   |
| :----- | :--- | :------------------------------------- |
| ない   | 否定 | 問題用紙を出さない人は採点しません。   |
| たい   | 愿望 | おもしろい小説が読みたいのです。       |
| う(よう) | 意志 | 来週、北京へ行こうと思っています。 |

## 五、形式体言

| 种类 | 用法   | 例句                                 |
| :--- | :----- | :----------------------------------- |
| の   | 体言化 | 音楽を聞くのが好きです。             |
| こと | 体言化 | 英語を話すことができます。           |
| つもり | 打算   | 来週、広州へ行くつもりです。         |
| ため | 目的   | 祖国を建設するためにがんばります。 |
