# 用法说明:

1. 拨音用「n」表示。如:
   <ruby>新<rt>あら</rt></ruby>聞 (sinbun)、<ruby>民<rt>たみ</rt></ruby>族 (minzoku)。

2. 促音将后面的辅音重写两个表示。如:
    * <ruby>国家<rt>こっか</rt></ruby> (kokka)、<ruby>雑<rt>ざっ</rt></ruby>誌 (zassi)。

3. 长音在元音之上加长音符号「-」表示。如:
    <ruby>習慣<rt>しゅうかん</rt></ruby> (syūkan)、<ruby>勝利<rt>しょうり</rt></ruby> (syōri)。

## 二、四种句子的现在式和过去式

| 式 / 句     | 现在式 (肯定)              | 现在式 (否定)                          | 过去式 (肯定)              | 过去式 (否定)                              |
| :---------- | :------------------------- | :------------------------------------- | :------------------------- | :----------------------------------------- |
| 断定句      | 本です                     | 本ではありません<br>本じゃありません     | 本でした                   | 本ではありませんでした<br>本じゃありませんでした |
| 描写句      | 赤いです<br>立派です   | 赤く(は)ありません<br>立派ではありません<br>立派じゃありません | 赤かったです<br>立派でした   | 赤く(は)ありませんでした<br>立派ではありませんでした<br>立派じゃありませんでした |
| 存在句      | あります<br>います       | ありません<br>いません                   | ありました<br>いました       | ありませんでした<br>いませんでした             |
| 叙述句      | 読みます<br>読んでいます | 読みません<br>読んでいません           | 読みました<br>読んでいました | 読みませんでした<br>読んでいませんでした         |

## 三、助词

| 种类     | 用法     | 例句                                       |
| :------- | :------- | :----------------------------------------- |
| 格助词「を」 | 表示宾语 | 新聞を読みます。                           |
|          | 移动地点 | 川が公園の中を流れています。                 |
| 格助词「と」 | 动作对手 | 友達といっしょに公園へ行きます。             |
| 格助词「で」 | 场所     | 食堂で食事をします。                       |
|          | 范围     | この店は上海で一番大きいです。               |
|          | 状态     | 二人で食事をします。                       |
|          | 方法     | 自転車で町へ行きます。                     |
