# 单元四复习

## (2) 各活用形的用法

### ① 未然形

动词未然形不能单独使用，必须后续助动词。如后续助动词「<ruby>う<rt>う</rt></ruby>」「<ruby>よう<rt>よう</rt></ruby>」「<ruby>ない<rt>ない</rt></ruby>」等。

*   <ruby>徐<rt>じょ</rt></ruby>さんは自転車を買<ruby>おう<rt>おう</rt></ruby>と思っています。
*   <ruby>講義<rt>こうぎ</rt></ruby>がわからない<ruby>時<rt>とき</rt></ruby>はどうしますか。

### ② 连用形

a 名词法：动词连用形作名词用。

*   <ruby>答<rt>こた</rt></ruby>える→<ruby>答<rt>こた</rt></ruby>え、<ruby>考<rt>かんが</rt></ruby>える→<ruby>考<rt>かんが</rt></ruby>え、<ruby>休<rt>やす</rt></ruby>む→<ruby>休<rt>やす</rt></ruby>み等。

b 连用法：以连用形直接和其他用言结合，构成复合词。

*   <ruby>書<rt>か</rt></ruby>き＋<ruby>終<rt>お</rt></ruby>わる→<ruby>書<rt>か</rt></ruby>き<ruby>終<rt>お</rt></ruby>わる、<ruby>作<rt>つく</rt></ruby>り＋<ruby>易<rt>やす</rt></ruby>い→<ruby>作<rt>つく</rt></ruby>り<ruby>易<rt>やす</rt></ruby>い、
    <ruby>喜<rt>よろこ</rt></ruby>び＋<ruby>合<rt>あ</rt></ruby>う→<ruby>喜<rt>よろこ</rt></ruby>び<ruby>合<rt>あ</rt></ruby>う、<ruby>入<rt>い</rt></ruby>れ＋<ruby>過<rt>す</rt></ruby>ぎる→<ruby>入<rt>い</rt></ruby>れ<ruby>過<rt>す</rt></ruby>ぎる。

c 后续各种助词，如「<ruby>て<rt>て</rt></ruby>」「<ruby>ても<rt>ても</rt></ruby>」「<ruby>たり<rt>たり</rt></ruby>」「<ruby>ながら<rt>ながら</rt></ruby>」等。
d 后续各种助动词，如「<ruby>たい<rt>たい</rt></ruby>」「<ruby>そうだ<rt>そうだ</rt></ruby>」(样态)「<ruby>ます<rt>ます</rt></ruby>」「<ruby>た<rt>た</rt></ruby>」等。

### ③ 终止形

a 直接用于结句，构成简体句。
b 后续各种助词，如「<ruby>と<rt>と</rt></ruby>」「<ruby>が<rt>が</rt></ruby>」「<ruby>から<rt>から</rt></ruby>」「<ruby>し<rt>し</rt></ruby>」「<ruby>か<rt>か</rt></ruby>」等。
c 后续各种助动词，如「<ruby>だろう<rt>だろう</rt></ruby>」「<ruby>でしょう<rt>でしょう</rt></ruby>」等。

### ④ 连体形

a 后续体言及形式体言「<ruby>の<rt>の</rt></ruby>」「<ruby>こと<rt>こと</rt></ruby>」「<ruby>つもり<rt>つもり</rt></ruby>」等。

*   <ruby>見<rt>み</rt></ruby>るのは<ruby>好<rt>す</rt></ruby>きですが、<ruby>や<rt>や</rt></ruby>るのはあまり<ruby>好<rt>す</rt></ruby>きではありません。

b 后续各种助词，如「<ruby>ので<rt>ので</rt></ruby>」「<ruby>だけ<rt>だけ</rt></ruby>」等。

*   <ruby>午後<rt>ごご</rt></ruby>、<ruby>用事<rt>ようじ</rt></ruby>があるので、くることができません。

### ⑤ 假定形

后续接续助词「<ruby>ば<rt>ば</rt></ruby>」。

### ⑥ 命令形

直接用于结句，表示命令。

*   <ruby>止<rt>と</rt></ruby>まれ。
*   <ruby>注意<rt>ちゅうい</rt></ruby>しろ。
*   <ruby>進<rt>すす</rt></ruby>め。
