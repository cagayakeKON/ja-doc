# 第3课

## 五、仿照例句改换讲法。

例：へやにつくえがあります。いすもあります。
→へやにつくえやいすなどがあります。

1.  つくえのうえにりんごがあります。レモンもあります。
2.  ベッドのうえにふとんがあります。ふくもあります。
3.  へやにテレビがあります。パソコンもあります。
4.  テーブルのうえにスプーンがあります。はしもあります。
5.  しんしつにつくえがあります。ベッドもあります。

## 六、用括弧里的词回答问题。

1.  ここはせんせいのクラブですか。(がくせいのクラブ)
2.  ここはあなたのへやですか。(李さんのへや)
3.  そこになにがありますか。(ほん)
4.  そこにすうがくのほんがありますか。(はい)
5.  そこにれきしのほんもありますか。(いいえ)
6.  れきしのほんはどこにありますか。(ほんだな)
7.  あなたのしんしつにベッドがいくつありますか。(みっつ)
8.  ほんだなもみっつありますか。(ひとつ)
