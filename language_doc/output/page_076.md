# (2) 一段动词的基本形和连用形

一段动词的词尾都是两个假名,最后一个假名都是「る」。词尾的头一个假名在イ段上的叫做上一段动词,在工段上的叫做下一段动词。词尾的头一个假名在ア行的叫做「ア行上(下)一段动词」,其他依此类推。

一段动词的词干和词尾有的能截然分开,有的不能。如「起きる」的词干是「お」,词尾是「きる」,而「見る」的「み」既是词干也是词尾。

一段动词词尾只要去掉最后一个假名「る」,就变成了连用形。
