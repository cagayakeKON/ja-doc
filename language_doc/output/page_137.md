# 第9课

## 谓语词组。

对象语是表示感觉、情感、巧拙、愿望、拥有、需要、可能等的句子成分。表示人的好恶或水平高低时，感情或能力所及的对象用「が」表示，具有这种感情或能力的人用「は」表示。

*   彼は<ruby>納豆<rt>なっとう</rt></ruby>や<ruby>梅干<rt>うめぼし</rt></ruby>が<ruby>嫌<rt>きら</rt></ruby>いです。(他讨厌纳豆和梅子。)
*   王さんは<ruby>料理<rt>りょうり</rt></ruby>を<ruby>作<rt>つく</rt></ruby>るのが<ruby>下手<rt>したて</rt></ruby>です。(小王做菜不行。)

「･･･が･･･」除了构成对象语和谓语词组外，还能构成主谓词组。例如：

*   わたしは<ruby>目<rt>め</rt></ruby>がよくありません。(我眼睛不好。)
*   上海は冬が<ruby>寒<rt>さむ</rt></ruby>いです。(上海冬天寒冷。)

## 二、BよりAのほうが･･･です

[Image: A graphic showing "テニスよりバドミントンのほうがすこし上手です。"]

这个句型表示“与B相比A更･･･”。「より」是格助词，表示比较的对象。「……」部分由形容词、形容动词或表示状态的词构成。

*   日本より中国のほうが<ruby>広<rt>ひろ</rt></ruby>いです。(中国比日本大。)
*   スリラー映画より人間の<ruby>暖<rt>あたた</rt></ruby>かい<ruby>心<rt>こころ</rt></ruby>を<ruby>伝<rt>つた</rt></ruby>える映画のほうが<ruby>好<rt>す</rt></ruby>きです。
    (与惊险片相比，我更喜欢传递人间温暖的电影。)
*   昨日より今日のほうが<ruby>暑<rt>あつ</rt></ruby>いです。(今天比昨天热。)

## 三、形式体言「の」和「こと」

[Image: A text box containing "野球は見ることは好きですが、自分でやることはあまり好きでは ありません。"]
[Image: A text box containing "李さんは音楽を聞くのが好きです。"]

用言和一个句子不能后续助词作主语、宾语等，这时就需要使之体言化。需要把某个用言或一个句子当作体言，使之成为主题、主语、宾语、补语等时，就要在用言或句子后面加上形式体言 (形式体言)。
<ruby>けいしきたいげん</ruby>
在句中起和体言相同的语法作用、本身没有实质意义的词叫形式体言。形式体言接在体言或用言连体形后，构成具有体言机能的词团。凡是能接在体言后面的助词基本上都可以接在词团后面。

「こと」「の」是形式体言。「の」可以表示人，也可以表示事或物；「こと」一般表示事情、事实、内容等含义。

*   王さんの家へ<ruby>行<rt>い</rt></ruby>ったのは<ruby>先週<rt>せんしゅう</rt></ruby>の<ruby>何曜日<rt>なんようび</rt></ruby>でしたか。(去小王家是上星期几？)
*   <ruby>大<rt>おお</rt></ruby>きいのは二元で、<ruby>小<rt>ちい</rt></ruby>さいのは一元です。(大的二元，小的是一元。)
