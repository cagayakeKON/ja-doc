# 九、AはBより…です

テニスはバドミントンよりすこし難しいです。

この句型表示“甲比乙……”。

*   中国は日本より広いです。(中国比日本大。)
*   今日は昨日より暑いです。(今天比昨天热。)
*   駅の前は公園よりにぎやかです。(车站前比公园热闹。)

# 十、動詞作定語的形式

「人間の暖かい心を伝える映画が好きです。」
「焼いた魚と煮た魚は好きです。」

私たちは、動詞の継続体で名詞を修飾する方法を学びました（第八課参照）。継続体のほか、動詞の現在形も名詞を修飾できます。「伝える」は動詞の現在形が名詞を修飾する例です。

動詞の現在形が名詞を修飾する場合、連体形を用いますが、その活用形と基本形は同じです。

*   乗るとき（乗る時） 食べるもの（食べる物）
*   来る人（来る人）

動詞の現在形が名詞を修飾するだけでなく、動詞の過去形も名詞を修飾できます。「焼いた魚」は動詞の過去形が名詞を修飾する例です。動詞の過去形が名詞を修飾する場合、連用形と過去の助動詞「た」を組み合わせて、「連用形 + た」の形になります。五段動詞が「た」の連用形と「て」の連用形が同じように活用しますが、音便があります。ガ行・イ音便、ナ・バ・マ行の撥音便が「た」に続く場合、「だ」に濁ります。例：

*   読んだ本（読んだ本） 見た映画（見た映画）
*   習った単語（習った単語）
*   わたしが乗るバスは上海駅から来ます。(我要乘的公共汽车是从上海站开来的。)
*   わたしが昨日訪ねた友達は今日北京へ帰りました。(我昨天拜访的那位朋友今天回北京了。)
*   これは昨日読んだ雑誌です。(这是昨天看过的杂志。)
