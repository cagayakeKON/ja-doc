# 五、･･･へ･･･を･･･に行きます(来ます)

あさっての晩、近くの文化館へ映画を見に行きませんか。

これは、来去目的を示す型です。「へ」は来去の場所、「に」は来去の目的を示します。目的の部分は「動詞のます形+に」または「漢語サ変動詞の語幹+に」で、「行く」「来る」「帰る」「もどる」などの移動を表す動詞が続きます。

*   レストランへフランス料理を食べに行きます。(去西餐馆吃法国菜。)
*   デパートへ買い物に行きました。(去百货商店买了东西。)
*   なにをしに上海へ来ましたか。(来上海做什么呢?)

# 六、并列助词「か」表示选择

今年は友達に中国のお茶かお酒をあげるつもりです。

文中の「か」は並列助詞で、名詞または活用詞の終止形の後につき、例の中から一つを選ぶことを示します。

*   授業が終わってから、わたしはたいてい教室か閲覧室で勉強します。
    (上完课,我一般在教室或阅览室学习。)
*   わたしか弟がもらいに来ます。(由我或者我弟弟来拿。)
*   あなたが来るかわたしが行くかです。(是你来或是我去。)
*   その仕事は王さんかだれかに頼むつもりです。(那工作我打算拜托小王或其他什么人。)

# 七、…(た)ことがあります

李さんもお姉さんに何か差し上げたことがありますか。

「…(た) ことがあります」は、ある経験があることを示し、否定形は「…(た)ことはありません」となります。「ことがあります」の前につく動詞は過去形を使います。

*   わたしは富士山に登ったことがあります。(我爬过富士山。)
*   海で泳いだことがあります。(曾在大海里游过泳。)
*   聞いたことはありますが、見たことはありません。(听说过,但没见到过。)

# 八、数量词+も

わたしはただもらうだけで、姉には一度もあげたことがありません。
