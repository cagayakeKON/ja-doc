# 七、格助詞「が」用于客观描述

<p>観光客が多かったですね。</p>

<p>格助詞「が」可用于对客观事物的描述。当被描述的事物在句中成为主语时用「が」表示。</p>

<ul>
  <li><ruby>本<rt>ほん</rt></ruby>が多いですね。(书真多啊。)</li>
  <li><ruby>山<rt>やま</rt></ruby>が高いですね。(山真高啊。)</li>
  <li><ruby>花<rt>はな</rt></ruby>がきれいですね。(花真美啊。)</li>
  <li><ruby>人<rt>ひと</rt></ruby>が少ないですね。(人真少啊。)</li>
</ul>

# 八、「これ、それ、あれ」「この、その、あの」的承上启下作用

<p>事物指示代词「これ、それ、あれ」和连体词「この、その、あの」除了指示现场实物外,还能指示会话中出现的事物。一般「これ、この」指示讲话人刚讲过或接下来就要讲述的事物,「それ、その」指对方刚讲过的事物,「あれ、あの」所指的事物是双方都了解,一提就明白的事物。另外,讲话人在讲话中指示自己刚讲过的事物时,如果断定这是听话人了解的事物,要用「そ」系指示词。例如:</p>

<p>A 友達のパーティーもありました。にぎやかで楽しかったです。そして、<ruby>そ<rt>そ</rt></ruby>の日<ruby>（<rt>（</rt></ruby>ちょうどわたしの誕生日でした。(有朋友的派对,非常热闹、愉快。而且,那天正好是我的生日。)</p>

<p>B <ruby>それ<rt>それ</rt></ruby>はよかったですね。(那不错啊。)</p>

# 九、接头词「お」「ご」

<p>「お」「ご」是接头词。接在表示和尊长有关的人、属于尊长所有的东西、叙述尊长的动作或状态的词前面,表示尊重,也用于表示恭敬。</p>

<p>一般「お」接在日本固有的词汇或日本式的汉语词汇前面,「ご」接在汉语词汇前面(有例外)。例如:</p>

<ul>
  <li>先生、お電話です。(老师,您的电话。)</li>
  <li>これは先生のお荷物ですか。(这是老师您的行李吗?)</li>
  <li>ご親切ありがとうございました。(谢谢您的热情帮助。)</li>
</ul>
