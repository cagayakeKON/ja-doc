# 解説

## 一、表示欲望的「ほしい」和「ほしがる」

友達は英語の辞書を<ruby>ほ<rt>ほ</rt></ruby>しがっていますが、わたしは<ruby>ほ<rt>ほ</rt></ruby>しくありません。
わたしは中国語の辞書が<ruby>ほ<rt>ほ</rt></ruby>しいのです。

「<ruby>ほ<rt>ほ</rt></ruby>しい」和「<ruby>ほ<rt>ほ</rt></ruby>しがる」是表示企求（想要某东西的企求）的词。「<ruby>ほ<rt>ほ</rt></ruby>しい」是形容词，一般表示讲话人的企求，问句也用于第二人称。想要的东西在
句中构成对象语，用「が」表示。「<ruby>ほ<rt>ほ</rt></ruby>しがる」是ラ行五段他动词，一般用于
叙述第三人称的企求。想要的东西在句中构成宾语，用「を」表示。「<ruby>ほ<rt>ほ</rt></ruby>しがっ
ている」表示现在的愿望。

*   誕生日に何が<ruby>ほ<rt>ほ</rt></ruby>しいですか。（过生日你想要什么礼物？）
*   一週間ぐらいの休みが<ruby>ほ<rt>ほ</rt></ruby>しいですね。（真想要一星期左右的休假。）
*   山田さんは大きい家を<ruby>ほ<rt>ほ</rt></ruby>しがっています。（山田想要有个大点的房子。）

一般来说，日本人为了表示对他人的尊重，不常用明显的表达方式表
达听者的内心想法。尤其表示企求或愿望的「<ruby>ほ<rt>ほ</rt></ruby>しい」及后面讲到的「<ruby>たい<rt>たい</rt></ruby>」
和表示意志的「つもり」，这种倾向更为明显。因此，日本人常常会用「…は
いかがですか」 「△△さんは…が<ruby>ほ<rt>ほ</rt></ruby>しいと言っています」的表达方式询问对方
的企求，或表示第三人称的企求和愿望。

## 二、表示希望的助动词「たい」和「たがる」

[Image: A dialogue box showing a question and answer in Japanese.]

なにが<ruby>買<rt>か</rt></ruby>いたいんですか。
（牧野さんは）ラジカセを<ruby>買<rt>か</rt></ruby>いたがっています。
ラジカセを<ruby>買<rt>か</rt></ruby>って中国語のラジオ講座を<ruby>聞<rt>き</rt></ruby>きたいと<ruby>思<rt>おも</rt></ruby>っています。

「<ruby>たい<rt>たい</rt></ruby>」「<ruby>た<rt>た</rt></ruby>がる」是表示愿望的助动词，接在动词连用形后。「<ruby>たい<rt>たい</rt></ruby>」
是形容词型活用，表示讲话人想做某件事，问句可用于第二人称，后续体
言作定语时可用于第二、第三人称。「<ruby>た<rt>た</rt></ruby>がる」是ラ行五段活用，表示讲话
人观察到的别人流露在外表或行为上的愿望。

「<ruby>たい<rt>たい</rt></ruby>」的对象语用「を」或「が」表示。用「を」时句子重点在谓语上，
用「が」时句子重点在对象语上，或者表示本能的愿望。
当用敬体与人说话时请注意尽量不要用「…<ruby>たい<rt>たい</rt></ruby>ですか」这种直接的询
问方式。
