# 四、词汇

(1) 日语词汇的来源大致有三类。

① 日语原有的词汇，日语叫做「<ruby>和<rt>わ</rt><ruby>語」如：
<ruby>逢<rt>あ</rt></ruby>う（遇见） <ruby>秋<rt>あき</rt></ruby>（秋天） <ruby>声<rt>こえ</rt></ruby>（声音） <ruby>汗<rt>あせ</rt></ruby>（汗）
<ruby>高<rt>たか</rt></ruby>い（高） <ruby>私<rt>わたし</rt></ruby>（我）

② 汉语词汇，日语叫「<ruby>漢<rt>かん</rt></ruby>語」，有直接采用汉语的和利用汉字创造出来的两种。
*   直接采用汉语的：学校 先生
*   利用汉字创造的：<ruby>切<rt>き</rt></ruby>手 <ruby>時<rt>と</rt></ruby>計
*   ③ 外来语词汇，日语叫「<ruby>外<rt>がい</rt></ruby><ruby>来<rt>らい</rt></ruby><ruby>語<rt>ご</rt></ruby>」，主要从欧美语言（多数是英语）音译而来。如：
    ノート インク テレビ ラジオ パン

(2) 日语汉字的音、形、义

① 读音
<ruby>音<rt>おん</rt></ruby>読<ruby>読<rt>ど</rt></ruby>（日语称「<ruby>音<rt>おん</rt></ruby><ruby>読<rt>ど</rt></ruby>」）和「<ruby>訓<rt>く</rt></ruby><ruby>読<rt>ど</rt></ruby>」（日语称「<ruby>訓<rt>く</rt></ruby><ruby>読<rt>ど</rt></ruby>」）两种读法。模仿汉语读音的叫做“<ruby>音<rt>おん</rt></ruby><ruby>読<rt>ど</rt></ruby>”，借用汉字的形和义而按日本固有语言来读的叫做“<ruby>訓<rt>く</rt></ruby><ruby>読<rt>ど</rt></ruby>”。如：

    音读：<ruby>電<rt>でん</rt></ruby><ruby>話<rt>わ</rt></ruby> <ruby>先<rt>せん</rt></ruby><ruby>生<rt>せい</rt></ruby> <ruby>学<rt>がく</rt></ruby><ruby>生<rt>せい</rt></ruby>
    <ruby>訓<rt>く</rt></ruby><ruby>読<rt>ど</rt></ruby>：<ruby>庭<rt>にわ</rt></ruby> <ruby>草<rt>くさ</rt></ruby> <ruby>花<rt>はな</rt></ruby>

    同一个汉字，由于<ruby>音<rt>おん</rt></ruby><ruby>読<rt>ど</rt></ruby>和<ruby>訓<rt>く</rt></ruby><ruby>読<rt>ど</rt></ruby>的关系，或者由于一字多义等关系，往<ruby>仮<rt>ふ</rt></ruby><ruby>名<rt>が</rt></ruby>

    往有多种读音。为了便于阅读，有时在汉字上边用<ruby>仮<rt>か</rt></ruby><ruby>名<rt>な</rt></ruby>注音，这种注音日语叫做「<ruby>振<rt>ふ</rt></ruby><ruby>仮<rt>りが</rt></ruby><ruby>名<rt>な</rt></ruby>」或「ルビ」。如以汉字“<ruby>人<rt>ひと</rt></ruby>”为例：
    <ruby>民<rt>みん</rt></ruby> <ruby>一<rt>いち</rt></ruby><ruby>人<rt>にん</rt></ruby> <ruby>四<rt>よ</rt></ruby><ruby>人<rt>にん</rt></ruby> <ruby>あの<rt>あの</rt></ruby> <ruby>人<rt>ひと</rt></ruby>

    用汉字和<ruby>仮<rt>か</rt></ruby><ruby>名<rt>な</rt></ruby>构成一个词的时候，写在汉字后面的<ruby>仮<rt>か</rt></ruby><ruby>名<rt>な</rt></ruby>日语叫做「<ruby>送<rt>お</rt></ruby><ruby>仮<rt>が</rt></ruby><ruby>名<rt>な</rt></ruby>」。它对判断某些汉字的读音起着决定性的作用。例如「<ruby>明<rt>あ</rt></ruby>ける」应读「あ
    ける」、「<ruby>明<rt>あ</rt></ruby>るい」应读「あかるい」、「<ruby>明<rt>あ</rt></ruby>らか」应读「あきらか」。因此，无论读
    或写的时候，都要认真注意「<ruby>送<rt>おく</rt></ruby><ruby>仮<rt>が</rt></ruby><ruby>名<rt>な</rt></ruby>」。

    ② 字形
    现代日语中的汉字字形和我国现在通行的汉字字形相比，有的完全相
    同，有的不完全相同。此外，某些是模仿汉字的构字法（主要是会意法）
    创造的，日语叫「<ruby>国<rt>こく</rt></ruby><ruby>字<rt>じ</rt></ruby>」。

    | :--------------- | :---------------------------------------------------------------------- |
    | 完全相同的       | 椅子 教室 学校 先生 学生 <br> 日语汉字：<ruby>決<rt>けつ</rt></ruby> <ruby>変<rt>へん</rt></ruby> <ruby>対<rt>たい</rt></ruby> <ruby>団<rt>だん</rt></ruby> <ruby>辺<rt>へん</rt></ruby> <br> 汉语汉字：决 变 对 团 边 |
    | 大同小异的       | 日语汉字：<ruby>払<rt>ふ</rt></ruby> <ruby>竜<rt>りゅう</rt></ruby> <ruby>仮<rt>か</rt></ruby> <ruby>売<rt>ばい</rt></ruby> <ruby>欠<rt>けつ</rt></ruby> <br> 汉语汉字：拂 龙 假 卖 缺 |
    | 差异很大的       | <ruby>畑<rt>はたけ</rt></ruby>（旱田） <ruby>峠<rt>とうげ</rt></ruby>（山岭） <ruby>辻<rt>つじ</rt></ruby>（十字路口） |
    | 模仿创造的       |                                                                         |

③ 词义
日语中的汉字词义有的和汉语相同，但也有不少和汉语的词义不同的。
