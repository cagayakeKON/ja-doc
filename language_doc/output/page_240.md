# (2)･･･たり、･･･たりします (です)

表示两个以上状态的反复或交替出现,或表示从许多状态中举出若干状态。形容词要用连用形「かっ」后续「たり」。

*   部屋の中は暖かかったり、寒かったりです。(屋里一会儿热,一会儿冷的。)
*   このごろ天気はよかったり、悪かったりです。(这些日子天气时好时坏的。)
*   山に登っている間は晴れたり<ruby>曇<rt>くも</rt></ruby>ったり の天気でした。(爬山的时候天气时晴时阴。)

# (3)･･･だったり･･･だったりします (です)

体言、形容动词后接「たり」时用「だったり」的形式,意义与形容词连用形后续「たり」相同。

*   この辺りは静かだったり、にぎやかだったりです。(这一带有时宁静,有时热闹。)
*   休みの日は土曜日だったり、日曜日だったりです。(休息天有时是星期六,有时是星期天。)

# 五、体言+にとって

<ruby>言葉<rt>ことば</rt></ruby>の違いは<ruby>彼<rt>かれ</rt></ruby>らにとってあまり大きな<ruby>問題<rt>もんだい</rt></ruby>ではありません。

「･･･にとって」是词组。「体言+にとって」表示叙述的事项是对该体言而言的,相当于汉语的“对于･･･来说”。「にとって」前面的名词大都是有关人或单位的名词。

*   これはわたしにとって大切なものです。(这东西对我来说很重要。)
*   この<ruby>問題<rt>もんだい</rt></ruby>は三年生にとっても、とても<ruby>難<rt>むずか</rt></ruby>しいのです。(这个问题即使对三年级学生来说也是很难的。)

# 六、「･･･(する)前に」和「･･･(した)後で」

<ruby>中国<rt>ちゅうごく</rt></ruby>へ来る前に<ruby>中国<rt>ちゅうごく</rt></ruby>のことを知っていましたか。
<ruby>北京<rt>ペキン</rt></ruby>オリンピックや上海万博についてのテレビ番組を見たあとで、<ruby>中国<rt>ちゅうごく</rt></ruby>へ留学しようと思いました。

动词连体形作「前」的定语时要用现在式表示;动词连体形作「後」的定语时要用过去式表示。「･･･前に」表示“･･･之前”,「･･･後で」表示“……之后”。

*   料理を作る前に手を洗います。(在做饭前先洗手。)
*   行く前に電話で知らせました。(去之前打电话通知了他。)
*   ご飯を食べたあとで勉強します。(吃饭后学习。)
