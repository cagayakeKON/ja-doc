# 日语语法体系框架

本框架基于传统学校语法体系整理，将日语词汇分为**自立语**与**附属语**两大类。

---

## 一、 自立语（独立成句，有实际意义）

### 1. 体言 (无活用)
特点：在句中可作主语，没有词性变化。

*   **名词**：表示人、事、物的名称。（例：本、日本、水）
*   **代词**：指代人、物、场所、方向。

    **“こそあど”系列（指示代词）：**
    日语的指示代词根据距离说话人的远近分为：**近称 (こ)**、**中称 (そ)**、**远称 (あ)**、**不定称 (ど)**。

    | 分类 | 近称 (こ) | 中称 (そ) | 远称 (あ) | 不定称 (ど) | 说明 |
    | :--- | :--- | :--- | :--- | :--- | :--- |
    | **事物** | これ | それ | あれ | どれ | 这个、那个、哪个 |
    | **场所** | ここ | そこ | あそこ | どこ | 这里、那里、哪里 |
    | **方向** | こちら | そちら | あちら | どちら | 这边、那边、哪边 |
    | **连体** | この | その | あの | どの | 后接名词 (这个...) |
    | **样态** | こう | そう | ああ | どう | 这样、那样、怎样 |
    | **种类** | こんな | そんな | あんな | どんな | 这种、那种、哪种 |
    | **口语方向** | こっち | そっち | あっち | どっち | 这边 (更口语) |
*   **数词**：表示数量、顺序或性质。

    **1. 基数词：**

    | 数字 | 音读 (おんよみ) | 训读 (くんよみ) |
    | :--- | :--- | :--- |
    | **0** | **零 (れい)** | - |
    | **1** | **一 (いち)** | **一つ (ひとつ)** |
    | **2** | **二 (に)** | **二つ (ふたつ)** |
    | **3** | **三 (さん)** | **三つ (みっつ)** |
    | **4** | **四 (し / よん)** | **四つ (よっつ)** |
    | **5** | **五 (ご)** | **五つ (いつつ)** |
    | **6** | **六 (ろく)** | **六つ (むっつ)** |
    | **7** | **七 (しち / なな)** | **七つ (ななつ)** |
    | **8** | **八 (はち)** | **八つ (やっつ)** |
    | **9** | **九 (く / きゅう)** | **九つ (ここのつ)** |
    | **10** | **十 (じゅう)** | **十 (とお)** |

    **大数单位 (音读)：**
    *   **百**：百 (ひゃく)
    *   **千**：千 (せん)
    *   **万**：万 (まん)
    *   **亿**：億 (おく)
    *   **兆**：兆 (ちょう)

    **2. 助数词（量词）：**
    附在数词后使用。许多量词在与特定数字结合时会产生**促音变**、**浊音变**或有**特殊读音**。

    | 量词 | 主要对象 (详细举例) | 1-10 的具体读音 (重点标注音变/特殊读音) |
    | :--- | :--- | :--- |
    | **～人 (にん)** | **人** (老师、学生、外籍人士等) | **1人 (ひとり)**、**2人 (ふたり)**、3人 (さんにん)、**4人 (よにん)**... |
    | **～本 (ほん)** | **细长物** (铅笔、瓶子、伞、香蕉、树木、细长的电影胶片、河流) | **1本 (いっぽん)**、2本 (にほん)、**3本 (さんぼん)**、**6本 (ろっぽん)**、**8本 (はっぽん)**、**10本 (じゅっぽん)** |
    | **～枚 (まい)** | **薄平物** (纸、照片、邮票、盘子、衬衫、被褥、CD、硬币) | 1枚 (いちまい)、2枚 (にまい)、3枚 (さんまい)、4枚 (よんまい)... (无音变) |
    | **～冊 (さつ)** | **装订书籍** (书、杂志、笔记本、相册、字典) | **1冊 (いっさつ)**、2冊 (にさつ)、**8冊 (はっさつ)**、**10冊 (じゅっさつ)** |
    | **～匹 (ひき)** | **昆虫、兽类、鱼类** (猫、狗、兔子、蛇、虾、猴子) | **1匹 (いっぴき)**、2匹 (にひき)、**3匹 (さんびき)**、**6匹 (ろっぴき)**、**8匹 (はっぴき)**、**10匹 (じゅっぴき)** |
    | **～足 (そく)** | **成对的鞋袜** (鞋子、袜子、拖鞋、足袋) | **1足 (いっそく)**、2足 (にそく)、**3足 (さんぞく)**、**8足 (はっそく)**、**10足 (じゅっそく)** |
    | **～回 (かい)** | **次数、频率** (去过几次、练习几次、第几次) | **1回 (いっかい)**、6回 (ろっかい)、**8回 (はっかい)**、**10回 (じゅっかい)** |
    | **～台 (だい)** | **机械、车辆、电器** (汽车、电视、电脑、钢琴、洗衣机) | 1台 (いちだい)、2台 (にだい)、3台 (さんだい)... (无音变) |
    | **～歳 (さい)** | **年龄** (人的年龄、动植物的岁数) | **1歳 (いっさい)**、**8歳 (はっさい)**、**10歳 (じゅっさい)** |

    **3. 序数词：**
    表示事物的顺序。
    *   在数字前加“第”（第一、第二）。
    *   在数字或助数词后加“目”（番目、回目）。

### 2. 用言 (有活用)
特点：在句中可作谓语，有词尾变化（活用）。

#### （1） 动词
表示动作、作用、存在。

**动词活用种类详析：**

| 活用种类 | 判断标准 (辞书形/原形) | 活用规律概要 | 具体例词 |
| :--- | :--- | :--- | :--- |
| **五段活用** | 1. 词尾不是「る」的动词。<br>2. 词尾是「る」，但「る」前面的假名不在「い段」或「え段」上。 | 词尾在「あ、い、う、え、お」五个段上进行变化。 | 行く、書く、話す、持つ、死ぬ、呼ぶ、飲む、作る |
| **上一段活用** | 词尾是「る」，且「る」前面的假名在**「い段」**上。 | 活用时去掉词尾的「る」，词干保持不变。 | 見る、起きる、降りる、借りる、信じる |
| **下一段活用** | 词尾是「る」，且「る」前面的假名在**「え段」**上。 | 活用时去掉词尾的「る」，词干保持不变。 | 食べる、寝る、教える、開ける、受ける |
| **カ行变格活用** | 只有**「来る (くる)」**一个词。 | 读音发生特殊变化（こ、き、くる、くれ、こい）。 | 来る |
| **サ行变格活用** | 1. **「する」**。<br>2. **「名词 + する」**。 | 根据后续词不同，发生特殊变化（し、せ、さ、する、すれ）。 | する、勉強する、散歩する、料理する |

**【注意】 特殊五段动词（容易混淆）：**
有些动词虽然外形符合“一段动词”的标准（词尾为「る」且前一假名为「い/え」段），但实际上属于**五段活用**。常见的包括：
*   **帰る (かえる)**：回来/回去
*   **入る (はいる)**：进入
*   **走る (はしる)**：奔跑
*   **知る (しる)**：知道
*   **切る (きる)**：剪/切
*   **要る (いる)**：需要
*   **喋る (しゃべる)**：聊天

**六大活用形详析：**

日语动词的六种活用形是语法逻辑的核心。以下按各类动词的变形规则进行详细总结。

---

##### 1. 未然形 (みぜんけい)
**【主要用法】** 后接否定助动词「ない」、意志助动词「う/よう」、使役助动词「せる/させる」、被动助动词「れる/られる」。

| 动词种类 | 变形规则 | 具体示例 |
| :--- | :--- | :--- |
| **五段活用** | **否定**：词尾「う」段变**「あ」**段<br>**意志**：词尾「う」段变**「お」**段 | 書く → 書か(ない) / 書こ(う)<br>死ぬ → 死な(ない) / 死の(う) |
| **一段活用** | 去掉词尾的**「る」** | 食べる → 食べ(ない) / 食べ(よう) |
| **サ行变格** | **否定**：变**「し / せ / さ」** <br>**意志**：变**「し」** | する → し(ない) / せ(ず) / さ(せる)<br>する → し(よう) |
| **カ行变格** | 变**「こ」** (读音变，写法为来) | 来る(くる) → 来(こ)ない / 来(こ)よう |

---

##### 2. 连用形 (れんようけい)
**【主要用法】** 后接敬体「ます」、过去「た」、接续「て」、希望「たい」、中顿（句子中间停顿）。

| 动词种类 | 变形规则 | 具体示例 |
| :--- | :--- | :--- |
| **五段活用** | **1. 常规连用形** (接ます/たい)：词尾变**「い」**段<br>**2. 音便形** (接て/た)：如下表详述 | 書く → 書き(ます)<br>如下表详述 |
| **一段活用** | 去掉词尾的**「る」** | 食べる → 食べ(ます) / 食べ(た) / 食べ(て) |
| **サ行变格** | 变**「し」** | する → し(ます) / し(た) / し(て) |
| **カ行变格** | 变**「き」** (读音变，写法为来) | 来る(くる) → 来(き)ます / 来(き)た / 来(き)て |

**【重点】五段动词的音便规则 (接「て/た」时)：**
五段动词在连接「て/た」时，词尾会发生形态变化以方便发音，称为“音便”。

| 音便类型 | 词尾原形 | 变化规律 | 具体示例 (て/た) |
| :--- | :--- | :--- | :--- |
| **い音便** | **く、ぐ** | 变**「い」** (浊音词尾后接「で/だ」) | 書く → 書い**て** / 泳ぐ → 泳い**で** |
| **促音便** | **う、つ、る** | 变**「っ」** | 買う → かっ**て** / 待つ → まっ**て** / 作る → つくっ**て** |
| **拨音便** | **ぬ、ぶ、む** | 变**「ん」** (后接「で/だ」) | 死ぬ → しん**で** / 呼ぶ → よん**单** / 読む → よん**で** |
| **无音便** | **す** | 变**「し」** | 話す → はなし**て** / 出す → だし**て** |
| **特殊** | **行く** | 变**「いっ」** | 行く → いっ**て** / いっ**た** |

---

##### 3. 终止形 (しゅうしけい)
**【主要用法】** 用于结句（陈述句末尾），或后续助词「から」、「が」（虽然）等。

| 动词种类 | 变形规则 | 具体示例 |
| :--- | :--- | :--- |
| **所有动词** | 维持**「辞书形」** (原形) 不变 | 書く、食べる、する、来る |

---

##### 4. 连体形 (れんたいけい)
**【主要用法】** 后接名词（体言）进行修饰，或后续形式名词「こと」、「もの」、「とき」。

**【注意】** 在传统的六大活用体系中，动词的“连体形”在形式上与“终止形（辞书形）”相同。但在实际运用中，作为**体言修饰语**（定语），动词的各种简体形式均可充当连体功能：

| 修饰形式 | 语法对应 | 具体示例 |
| :--- | :--- | :--- |
| **原型 (肯定)** | 连体形 (辞书形) | 食べる人 (吃的人/将要吃的人) |
| **否定简体** | 未然形 + 助动词「ない」 | 食べ**ない**人 (不吃的人) |
| **过去简体** | 连用形 (音便) + 助动词「た」 | 食べ**た**人 (吃了的人) |
| **过去否定简体** | 未然形 + 助动词「なかった」 | 食べ**なかった**人 (没吃过的人) |

**总结：** 只要是**简体形式（普通体）**，无论是肯定、否定还是过去式，都可以直接放在体言前做修饰语。

---

##### 5. 假定形 (かていけい)
**【主要用法】** 后接接续助词「ば」，表示“如果...的话”。

| 动词种类 | 变形规则 | 具体示例 |
| :--- | :--- | :--- |
| **五段活用** | 词尾「う」段变**「え」**段 + ば | 書く → 书け(ば) / 呼ぶ → 呼べ(ば) |
| **一段活用** | 去掉词尾的「る」 + **「れ」** + ば | 食べる → 食べれ(ば) |
| **サ行变格** | 变**「すれ」** + ば | する → すれ(ば) |
| **カ行变格** | 变**「くれ」** + ば | 来る(くる) → 来(くれ)ば |

---

##### 6. 命令形 (めいれいけい)
**【主要用法】** 表示强硬的命令语气。

| 动词种类 | 变形规则 | 具体示例 |
| :--- | :--- | :--- |
| **五段活用** | 词尾「う」段变**「え」**段 | 書く → 書け！ / 泳ぐ → 泳げ！ |
| **一段活用** | 去掉词尾的「る」 + **「ろ」** (口语) / **「よ」** (文章) | 食べる → 食べろ！ / 見る → 見よ！ |
| **サ行变格** | 变**「しろ」** (口语) / **「せよ」** (文章) | する → しろ！ / 勉強せよ！ |
| **カ行变格** | 变**「こい」** | 来る(くる) → 来(こ)い！ |

#### （2） 形容词
表示性质、状态，词尾以**「い」**结束。（例：高い、美しい、悪い）

**形容词六大活用形详析：**

形容词的活用全部集中在词尾的「い」上。

---

##### 1. 未然形 (みぜんけい)
**【变形规则】** 词尾「い」变为 **「かろ」**。
**【主要用法】** 后接推量助动词「う」，表示对某种性质或状态的推测。
*   **示例**：高い → 高**かろ**う (大概很贵吧)
*   **示例**：寒い → 寒**かろ**う (大概很冷吧)
*   **注意**：这种形式在现代日语口语中已较少直接使用，多被「高いだろう」取代，但在书面语中依然可见。

---

##### 2. 连用形 (れんようけい)
形容词的连用形有两种主要形式，承担着极其丰富的功能。

**① 形式一：词尾「い」变为 「かっ」**
*   **用途**：后接过去助动词**「た」**。
*   **示例**：暑い → 暑**かっ**た (很热过/以前很热)
*   **示例**：新しい → 新し**かっ**た (以前很新)

**② 形式二：词尾「い」变为 「く」**
*   **用途 A (否定)**：后接否定助动词「ない」。
    *   例：遠い → 远**く**ない (不远)
*   **用途 B (中顿/接续)**：后接接续助词「て」，表示原因或并列。
    *   例：安**く**て、いい。 (又便宜又好。)
*   **用途 C (副词化)**：修饰后面的动词。
    *   例：速**く**走る (快速奔跑) / 强**く**なる (变强)

---

##### 3. 终止形 (しゅうしけい)
**【变形规则】** 维持原形（辞书形）不变。
**【主要用法】**
1.  **结句**：表示陈述事实。
    *   例：この花は美し**い**。 (这朵花很美。)
2.  **后续助词**：后接「から(因为)」、「が(虽然)」、「し(既...又)」。
    *   例：面白**い**から、読む。 (因为有趣，所以读。)

---

##### 4. 连体形 (れんたいけい)
**【变形规则】** 维持原形（辞书形）不变。
**【主要用法】** 直接放在体言（名词）前进行修饰。
*   **示例**：広**い**部屋 (宽敞的房间)
*   **示例**：詳し**い**説明 (详细的说明)
*   **注意**：在功能上，不仅原形可以修饰名词，形容词的过去式「～かった」和否定形式「～くない」同样可以修饰名词（如：暑かった日、赤くない服）。

---

##### 5. 假定形 (かていけい)
**【变形规则】** 词尾「い」变为 **「けれ」**。
**【主要用法】** 后接接续助词「ば」，表示假设条件。
*   **示例**：良**けれ**ば (如果好的话)
*   **示例**：美味し**けれ**ば (如果好吃的话)

---

##### 6. 命令形 (めいれいけい)
**【说明】** 现代描述性的形容词通常**没有命令形**。但在部分古语残留或极其特殊的情况下（如「よかれ」），虽具命令形式，但多表达祈愿。在一般语法学习中，记为“无”即可。

**【补充】** 
1.  **「いい」的特殊性**：形容词「いい」(好) 在进行活用时，必须使用其古语形式**「よい」**作为词干。
    *   否定：よ**く**ない (不可说いけない)
    *   过去：よ**かっ**た (不可说いかった)
2.  **推量表达**：虽然未然形「～かろう」在文法上存在，但在现代口语中，更常用**「终止形 + だろう/でしょう」**（例：高いだろう）来表示推测。

#### （3） 形容动词
表示性质、状态，词尾以**「だ」**结束（例：静かだ、綺麗だ、賑やかだ）。在表达上具有名词的稳定性，但在活用上类似用言。

**形容动词六大活用形详析：**

形容动词的活用主要体现在词尾的「だ」上。

---

##### 1. 未然形 (みぜんけい)
**【变形规则】** 词尾「だ」变为 **「だろ」**。
**【主要用法】** 后接推量助动词「う」，表示猜测。
*   **示例**：好きだ → 好き**だろ**う (大概喜欢吧)
*   **示例**：暇だ → 暇**だろ**う (大概有空吧)

---

##### 2. 连用形 (れんようけい)
形容动词的连用形有三种形式，对应不同的语法功能：

**① 形式一：词尾「だ」变为 「だっ」**
*   **用途**：后接过去助动词**「た」**。
*   **示例**：綺麗だ → 綺麗**だっ**た (以前很漂亮)
*   **示例**：有名だ → 有名**だっ**た (以前很有名)

**② 形式二：词尾「だ」变为 「で」**
*   **用途**：表示中顿、并列或后接否定。
*   **示例**：静か**で**、いい所だ。 (又安静，又是好地方。)
*   **示例**：好き**で**はない。 (并不喜欢。)

**③ 形式三：词尾「だ」变为 「に」**
*   **用途**：副词化，修饰后面的动词。
*   **示例**：静か**に**歩く (安静地走)
*   **示例**：上手**に**なる (变得擅长)

---

##### 3. 终止形 (しゅうしけい)
**【变形规则】** 维持原形 **「だ」** 不变。
**【主要用法】** 结句，表示陈述。
*   **示例**：この町は賑やか**だ**。 (这个城市很热闹。)
*   **注意**：在礼貌体中，终止形的词尾会变为「です」。

---

##### 4. 连体形 (れんたいけい)
**【变形规则】** 词尾「だ」变为 **「な」**（这是形容动词最核心的区别点）。
**【主要用法】** 修饰名词（体言）。
*   **示例**：親切**な**人 (亲切的人)
*   **示例**：特別**な**日 (特别的日子)

---

##### 5. 假定形 (かていけい)
**【变形规则】** 词尾「だ」变为 **「なら」**。
**【主要用法】** 表示假设条件。
*   **示例**：嫌い**なら**(ば)、食べなくてもいい。 (如果讨厌的话，不吃也可以。)
*   **注意**：现代口语中通常省略后面的「ば」。

---

##### 6. 命令形 (めいれいけい)
**【说明】** 形容动词通常**没有命令形**。但在极个别特殊文体或固定表达中（如「静かにしろ！」中的“静かに”实际上是连用形接动词命令形），形容动词本身不具备命令活用。

**【注意】** 
1.  **连体形「な」**：这是形容动词最重要的特征。因为它在修饰名词时必须变「だ」为「な」，所以常被称为“な形容词”。
2.  **特殊性**：形容动词在「终止形」和「连用形②」上与断定助动词「だ」的变形规律高度一致，但在「连体形」上则有其独特的「な」。

---

## 二、 附属语（不能独立成句，附着在自立语后）

### 1. 助动词 (有活用)
辅助用言或体言，增添陈述意义。助动词自身也具有活用变化。
**【常用助动词】**：
    *   **判断**：だ、です
    *   **敬体**：ます
    *   **否定**：ない、ぬ
    *   **过去/完了**：た
    *   **推量/意志**：う、よう
    *   **使役**：せる、させる
    *   **被动/可能/尊敬/自发**：れる、られる
    *   **希望**：たい、たがる
    *   **样态/传闻**：そうだ
    *   **比况**：ようだ、みたいだ
---

#### （1） 断定助动词：だ / です
用于表达“是...”。「だ」为简体（非正式），「です」为敬体（礼貌）。

**【接续规则】**
*   **名词 / 代词**：直接接续。 (例：学生だ / 私です)
*   **形容动词**：接在词干后。 (例：静かだ / 綺麗です)
*   **形容词**：
    *   **简体**：**不可接「だ」**。形容词原形本身即可结句。 (正：高い。 误：高いだ。)
    *   **敬体**：直接在形容词原形或过去式后加**「です」**以示礼貌。 (例：高いです / 高かったです)

**【“だ / です” 的常用时态变化对照表】**
针对名词及形容动词（词干），其简体与敬体的变化如下：

| 时态/语态 | **简体 (だ)** | **敬体 (です)** | 示例 (以“学生”为例) |
| :--- | :--- | :--- | :--- |
| **现在肯定** | **～だ** | **～です** | 学生**だ** / 学生**です** |
| **现在否定** | **～ではない** | **～ではありません** | 学生**ではない** / 学生**ではありません** |
| **过去肯定** | **～だった** | **～でした** | 学生**だった** / 学生**でした** |
| **过去否定** | **～ではなかった** | **～ではありませんでした** | 学生**ではなかった** / 学生**ではありませんでした** |
| **推测** | **～だろう** | **～でしょう** | 学生**だろう** / 学生**でしょう** |

**【注意】**
1.  **口语缩略**：在口语中，「では」经常会缩略为 **「じゃ」**。
    *   例：学生**ではない** → 学生**じゃない**
    *   例：学生**ではありません** → 学生**じゃありません**
2.  **形容词的敬体否定**：形容词变为敬体否定时有两种常用的表达方式：
    *   方式A：高**くないです** (更亲切、常用)
    *   方式B：高**くありません** (更正式、生硬)

---

#### （2） 敬体助动词：ます
接在**动词的连用形**后，表示对听话者的礼貌、尊敬语气。

**【接续规则：如何接“ます”】**
动词在连接“ます”前，必须先变为**连用形**：
*   **五段动词**：词尾「う」段变为**「い」**段（如：書く → 書き + ます）。
*   **一段动词**：去掉词尾的**「る」**（如：食べる → 食べ + ます）。
*   **サ行变格**：する 变为 **「し」**（如：勉強する → 勉強し + ます）。
*   **カ行变格**：来る 变为 **「き」**（如：来る → き + ます）。

**【“ます”的常用时态变化】**
虽然“ます”在语法上有六大活用，但在实际口语和书面语中，最常用的是以下四种时态形式：

| 时态/语态 | 形式 | 语法意义 | 具体示例 (以“書く”为例) |
| :--- | :--- | :--- | :--- |
| **现在肯定** | **～ます** | 表示习惯或将要发生的动作 | 書き**ます** (写 / 要写) |
| **现在否定** | **～ません** | 表示否定习惯或将不发生的动作 | 書き**ません** (不写 / 没打算写) |
| **过去肯定** | **～ました** | 表示动作已完成（过去式） | 書き**ました** (写了) |
| **过去否定** | **～ませんでした** | 表示过去没有发生动作 | 書き**ませんでした** (没写) |
| **提议/意志** | **～ましょう** | 表示建议或自己的意志 | 書き**ましょう** (写吧 / 我来写) |

**【活用小结】**
“ます”是一个非常特殊的助动词，它主要用来调整句子的礼貌程度。
*   其否定式 **「～ません」** 的活用来源于未然形。
*   其过去式 **「～ました」** 是连用形「まし」加助动词「た」构成的。
*   其提议式 **「～ましょう」** 来源于未然形「ませ」加上意志助动词「う」的音变形式。

---

#### （3） 否定助动词：ない
用于表示否定。

**【核心特征】**：其活用规律与**「形容词」**完全一致。

**【接续规则：不同词性如何变否定】**

| 词性种类 | 接续方式 | 具体变形规律 | 示例 |
| :--- | :--- | :--- | :--- |
| **五段动词** | 接**未然形** | 词尾「う」段变**「あ」**段 + ない | 行く → 行か**ない** / 読む → 读ま**ない** |
| **一段动词** | 接**未然形** | 去掉词尾的**「る」** + ない | 食べる → 食べ**ない** / 寝る → 寝**ない** |
| **サ行变格** | 接**未然形** | する 变为 **「し」** + ない | 勉強する → 勉強し**ない** |
| **カ行变格** | 接**未然形** | 来る (くる) 变为 **「こ」** + ない | 来る → **こ**ない (汉字写为“来”) |
| **形容词** | 接**连用形** | 词尾「い」变为 **「く」** + ない | 高い → 高**くない** / 寒い → 寒**くない** |
| **形容动词** | 接**连用形** | 词尾加 **「で(は / じゃ)」** + ない | 静かだ → 静か**ではない / じゃない** |
| **名词** | 接**连用形** | 加 **「で(后续强调语气)」** + ない | 学生 → 学生**ではない / じゃない** |

**【注意：特殊词否定】**
*   **「ある」(有)** 的否定形式不是“あらない”，而是直接使用形容词 **「ない」**。
*   **「いう」(说)** 的否定是 **「いわない」** (词尾是“う”的五段动词，未然形变“わ”而非“あ”)。

**【“ない” 的六大活用表】**
(此活用与其接续的对象无关，是指「ない」本身在句尾或句中时产生的变化)

| 活用形 | 形式 | 主要用法 | 具体示例 |
| :--- | :--- | :--- | :--- |
| **未然形** | なかろ(う) | 推测否定 | 行か**なかろう** (大概不去吧) |
| **连用形 ①** | なかっ(た) | 过去否定 | 行か**なかった** (没去) |
| **连用形 ②** | なく | 否定中顿/副词化 | 行か**なく**て / 行か**なく**なる |
| **终止形** | ない | 结句否定 | 行か**ない**。 (不去。) |
| **连体形** | ない | 修饰名词 | 行か**ない**人 (不去的人) |
| **假定形** | なけれ(ば) | 假设否定 | 行か**なければ** (如果不去的话) |

### 2. 助词 (无活用)
表示词与词之间的关系，或增添语气。

*   **格助词**：表示在句中的地位（が、を、に、へ、と、から、より、で、の）。
*   **副助词**：增添提示、程度、限度等意义（は、も、こそ、さえ、でも、だけ、しか）。
*   **接续助词**：连接句子（て、から、ので、のに、が、けれど）。
*   **终助词**：位于句末，表达情感、疑问或强调（か、ね、よ、な、わ、ぞ）。
